### Extraction tooling

New: `tools/captures/extract_a75_modes.mjs` — parses the official driver's JS bundle for any `ddeerXxxColour` class's `m_color_names` array and emits the catalog with derived wire codes (`max(0, idx - 2)` per the JS source). Reusable for adding other keyboard models. Usage:
```
node tools/captures/extract_a75_modes.mjs <bundle.js> [ddeerXxxColour]
```
Defaults to `ddeerA75ProColour`. All A75 / G65 / G75 / G60 / X60 family classes have identical 21-entry catalogs.
